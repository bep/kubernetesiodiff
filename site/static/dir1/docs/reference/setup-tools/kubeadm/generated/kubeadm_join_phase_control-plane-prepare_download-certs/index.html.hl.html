<h3 id="synopsis">
  Synopsis
</h3>
<p>
  [EXPERIMENTAL] Download certificates shared among control-plane nodes from the kubeadm-certs Secret
</p>
<pre><code>kubeadm join phase control-plane-prepare download-certs [api-server-endpoint] [flags]</code></pre>

  Options
</h3>

<p>

      &ndash;certificate-key string

  <tr>
    <td></td>

<p>
  <tr>
    <td colspan="2">
      &ndash;config string
    </td>

      Path to kubeadm config file.
    </td>
  </tr>
</p>
<p>
  <tr>
    <td colspan="2">
      &ndash;control-plane
    </td>
  </tr>
  <tr>
    <td></td>
    <td style="line-height: 130%; word-wrap: break-word;">
      Create a new control plane instance on this node
    </td>
  </tr>
</p>
<p>
  <tr>
    <td colspan="2">
      &ndash;discovery-file string
    </td>
  </tr>
  <tr>
    <td></td>
    <td style="line-height: 130%; word-wrap: break-word;">
      For file-based discovery, a file or URL from which to load cluster information.
    </td>
  </tr>
</p>
<p>
  <tr>
    <td colspan="2">
      &ndash;discovery-token string
    </td>
  </tr>
  <tr>
    <td></td>
    <td style="line-height: 130%; word-wrap: break-word;">
      For token-based discovery, the token used to validate cluster information fetched from the API server.
    </td>
  </tr>
</p>
<p>
  <tr>
    <td colspan="2">
      &ndash;discovery-token-ca-cert-hash stringSlice
    </td>
  </tr>
  <tr>
    <td></td>
    <td style="line-height: 130%; word-wrap: break-word;">
      For token-based discovery, validate that the root CA public key matches this hash (format: &ldquo;&lt;type&gt;:&lt;value&gt;&rdquo;).
    </td>
  </tr>
</p>
<p>
  <tr>
    <td colspan="2">
      &ndash;discovery-token-unsafe-skip-ca-verification
    </td>
  </tr>
  <tr>
    <td></td>
    <td style="line-height: 130%; word-wrap: break-word;">
      For token-based discovery, allow joining without &ndash;discovery-token-ca-cert-hash pinning.
    </td>
  </tr>
</p>

  <tr>
    <td colspan="2">
      -h, &ndash;help
    </td>
  </tr>
  <tr>
    <td></td>
    <td style="line-height: 130%; word-wrap: break-word;">
      help for download-certs
    </td>
  </tr>
</p>
<p>

    <td colspan="2">
      &ndash;tls-bootstrap-token string
    </td>
  </tr>
  <tr>
    <td></td>
    <td style="line-height: 130%; word-wrap: break-word;">
      Specify the token used to temporarily authenticate with the Kubernetes Control Plane while joining the node.

  </tr>

<p>
  <tr>
    <td colspan="2">
      &ndash;token string
